# peg-game

Simple peg jumping game ala the ones you find at Cracker Barrel.

## Installation and Usage

Follow the steps below to play:

<code>git clone https://github.com/bryanqb07/peg-game.git</code>  
<code>cd peg-game</code>  
<code>lein run</code>
